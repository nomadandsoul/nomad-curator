🔗 GitHub status webhook

Voeg deze Make.com HTTP module toe als laatste stap in elk scenario. Daarna synchroniseert de monitor automatisch bij elke uitvoering.

Make.com → HTTP → PUT naar GitHub
URL: https://api.github.com/repos/
  nomadandsoul/nomad-curator/
  contents/status.json

Method: PUT
Header: Authorization: Bearer [TOKEN]

Body:
{
 "scenario": "{{scenario_naam}}",
 "gescand": {{aantal_gescand}},
 "goedgekeurd": {{aantal_goed}},
 "timestamp": "{{now}}"
}
ℹ️ Maak een GitHub Personal Access Token aan via github.com → Settings → Developer settings → Tokens → repo scope
