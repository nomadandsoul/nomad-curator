Setup stappen — in volgorde uitvoeren
1
Higgsfield account + API key aanmaken
Vereist

Higgsfield gebruikt een KEY_ID + KEY_SECRET combinatie — niet één enkele key.

  1. Ga naar cloud.higgsfield.ai → maak een account aan
  2. Kies een abonnement (betaald — gratis tier heeft geen API-toegang)
  3. Ga in je dashboard naar API Keys
  4. Klik op Generate new key
  5. Kopieer de KEY_ID en KEY_SECRET apart — je ziet de secret maar één keer
Authenticatie formaat in Make.com:
Header naam: Authorization
Waarde: Key JOUW_KEY_ID:JOUW_KEY_SECRET
2
Buffer account + TikTok & Instagram koppelen
Vereist
  1. Ga naar buffer.com → maak een gratis account aan
  2. Klik op Connect channels → voeg TikTok toe → voeg Instagram toe
  3. Ga naar buffer.com/developers → maak een app aan
  4. Kopieer de Access Token
  5. Ga naar je Buffer dashboard → URL van jouw TikTok profiel bevat het profile ID
Profile ID vinden: Ga in Buffer naar je kanaalinstellingen. Het ID staat in de URL: buffer.com/profiles/PROFILE_ID/tab/queue
3
Google Sheet aanmaken voor content log
Vereist

Maak een nieuw Google Sheet aan via sheets.new met de naam Nomad & Soul — Content Log. Voeg rij 1 toe met deze headers:

A: Product ID B: Product naam C: ROAS score D: Merkfit score E: TikTok hook F: TikTok caption G: Instagram caption H: Video URL I: Collectie J: Seizoen K: Model stijl L: Datum gepost

Kopieer het Sheet ID uit de URL: docs.google.com/spreadsheets/d/DIT_IS_HET_ID/edit

4
Blueprint importeren in Make.com
Vereist
  1. Ga naar make.com → klik op + Create scenario
  2. Klik op het tandwiel-icoon linksboven in de scenario-editor
  3. Kies Import Blueprint
  4. Upload het bestand make-higgsfield-blueprint.json
  5. Het scenario verschijnt met 12 modules
5

6
Testen met één product
Vereist
  1. Zorg dat er minimaal één actief product in Shopify staat met een productfoto
  2. Klik in Make.com op Run once
  3. Controleer per module of de output correct is — groen vinkje = OK
  4. Na module 3: bekijk of Claude een score en content heeft gegenereerd
  5. Na module 7: kopieer de request_id en wacht 90 seconden
  6. Na module 9: controleer of status = COMPLETED en of er een video URL is
  7. Controleer Buffer of de post in de wachtrij staat
Tip: Zet Buffer eerst op Draft mode zodat posts niet automatisch worden gepubliceerd tijdens het testen. Schakel dit pas uit als alles werkt.
7
Inplannen op vaste momenten
Na testen

Zodra het testen geslaagd is, stel je een vaste planning in:

• Klik in Make.com op Schedule bovenaan het scenario
• Stel in: Elke dinsdag om 09:00
• Dit laat je maandag nieuwe producten goedkeuren in de curator app
• Dinsdag worden automatisch de video's gegenereerd en gepost
