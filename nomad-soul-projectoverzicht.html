10. Homepage Custom Liquid — Installatiehandleiding

Belangrijk: Altijd als .html bestand aanleveren en de HTML-inhoud plakken in de Shopify Custom Liquid sectie. Nooit als .liquid bestand.
1
Backup maken
Shopify Admin → Online Store → Thema's → Dupliceren
2
Code editor openen
Online Store → Thema's → Acties → Code bewerken
3
Sectie aanmaken
Map "Sections" → Nieuw bestand → naam: nomad-soul-homepage
4
Code plakken & opslaan
Volledige code uit .html bestand plakken → Opslaan
5
Sectie toevoegen
Thema's → Aanpassen → Sectie toevoegen → "Nomad & Soul Homepage"
6
Preview controleren
Desktop én mobiel checken — let op iOS Safari (background-attachment fix)
